## Hi, I'm Yizhi Mei👋

Ph.D. Candidate at Oita University, Japan
Researching software architecture, dependency analysis, and tooling for large-scale Vue.js applications.

---

## Research Interests

* Frontend Software Architecture
* Dependency Analysis for Vue.js Ecosystems
* Static Analysis and AST-based Tooling
* Configuration-Driven Frontend Engineering
* Scalable State Management Systems

---

## Current Research Projects

### VEXA

Configuration-driven frontend architecture framework for large-scale Vue.js applications.

* AST-based dependency analysis
* Explicit component-store relationship modeling
* Automated configuration-to-code generation
* Modular and demand-driven state management

### VueDependencyAnalyzer

Static dependency analysis tooling for Vue.js ecosystems.

* Component dependency extraction
* AST parsing pipelines
* Architecture visualization
* Frontend analyzability research

### Vuex Lazy Loading Framework

Research prototype for dynamic Vuex module lazy loading.

* Runtime module registration
* On-demand state loading
* Frontend initialization optimization
* Large-scale SPA maintainability
